React Hangman Game

Overview:
This project is a Hangman game built using React.
The application selects a random word and allows the user to guess letters to reveal the word.

If too many incorrect guesses are made, the hangman drawing is completed and the player loses.

How to Play:

1. Click letters on the keyboard to guess a word.
2. Correct guesses reveal letters in the word.
3. Incorrect guesses adds parts to the hangman drawing.
4. You must get the word before the drawing is completed to win.
5. If the drawing completes first, you lose.

Features:
Random word selection
Interactive keyboard
Dynamic Hangman drawing
Win / Lose detection
A restart game functionality
A help and rules section

Installation:
To run the project locally do the following:

1. Clone the repository
2. Navigate to the project folder
3. Install dependencies-

npm install

4. Start the React application-

npm start

The game will open in the browser at: http://localhost:3000
